# MARK-DOWN ENGINE VALIDATION DOCUMENT

This document is intentionally mixed with different structural patterns to test the

document intelligence, normalization, diagnostics, and suggestion engine.

---

## Overview

The system should correctly identify:

- headings

- commands

- diagrams

- tables

- procedures

- callouts

- roadmap phases

It should **NOT** incorrectly classify normal sentences as headings.

---

## Example Setup Guide Section

Install dependencies

pip install pandas numpy

pip install networkx scipy

Create project folder

mkdir helixa-pro

cd helixa-pro

Run application

python app.py

Expected output:

Server started at http://localhost:8501

## Database Schema Example

CREATE TABLE datasets (

    id INTEGER PRIMARY KEY,

    name TEXT,

    created_at TEXT

);

CREATE TABLE experiments (

    id INTEGER PRIMARY KEY,

    dataset_id INTEGER,

    score REAL

);

SELECT * FROM datasets;

# SOP - Sensor Calibration Protocol

Purpose
Define repeatable and safe execution steps for Sensor Calibration Protocol.

Scope
Applies to operators, reviewers, and supervisors.

Procedure
Prepare tools, verify prerequisites, and execute ordered actions.

Steps:
1. Confirm required materials and tools
2. Validate environmental readiness
3. Execute primary operation
4. Record measurements and outcomes
5. Escalate exceptions

Checklist:
- [ ] Pre-check completed
- [ ] Safety controls active
- [ ] Operation log captured

Validation:
- Verify measured values are within tolerance
- Confirm audit record exists

Safety Considerations
- Wear required protective equipment
- Stop immediately on abnormal condition

| Checkpoint | Criteria | Responsible |
|---|---|---|
| Pre-check | Tools verified | Operator |
| Execution | Procedure followed | Operator |
| Review | Record approved | Supervisor |
